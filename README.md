Setup:

1. Create a virtual environment using:
   ```
   pipenv shell
   ```
2. Install necessary libraries:
   ```
   pipenv install
   ```
3. Create an environmental variable file (.env) and link it.
4. Add your API keys for the following APIs:
   - OpenAI API
   - SerpAPI
5. Run the app.py file.
